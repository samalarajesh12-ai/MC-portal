# **App Name**: MARUTHI CLINIC PORTAL

## Core Features:

- Patient Data Loading: Fetch and display patient information.
- Appointment Scheduling: Allow patients to schedule, reschedule, or cancel appointments.
- Medication Refill Requests: Enable patients to request medication refills through the portal.
- Secure Messaging: Facilitate secure communication between patients and healthcare providers.
- Health Record Access: Provide patients with access to their health records, including lab results and medical history.
- AI-Powered Symptom Checker: Integrate a tool that allows patients to enter their symptoms and receive potential diagnoses and recommendations.

## Style Guidelines:

- Primary color: Dark teal (#338BA8), for a calm, professional, and trustworthy feel.
- Background color: Light grayish-blue (#E8F4F8). It's desaturated hue of the primary offers a clean and calming backdrop, allowing the content to stand out.
- Accent color: Blue-violet (#5B4281), this provides contrast and visual interest while remaining harmonious with the teal tones.
- Body and headline font: 'PT Sans', a humanist sans-serif, should be used throughout for a modern and readable experience. Its versatility ensures readability for both headings and body text.
- Use clear, modern icons to represent different sections and actions within the portal. Consistent style across all icons.
- Maintain a clean, organized layout with clear hierarchy. Use white space effectively to improve readability and reduce clutter.
- Subtle transitions and animations to enhance user experience (e.g., loading animations, hover effects). Keep animations consistent and avoid excessive use.
