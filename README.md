
# MARUTHI CLINIC PORTAL (v1.3.1)

A professional, cloud-synced clinic management system built with **Next.js + Firebase + ShadCN**.

## ☁️ Cross-Device Cloud Sync (New in v1.3.0)
This application is now fully integrated with **Firebase Firestore**. 
- **How it works**: Your data is no longer stored on your computer. It is stored in the Google Cloud.
- **Access Anywhere**: Simply copy the URL from your laptop's browser and open it on your mobile device.
- **Stay Logged In**: Log in with the same email/password on any device to see your synchronized appointments, medications, and bills.
- **Account Privacy**: Different accounts see different data. Your medical records are private and secured by your unique login.

## Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Firestore (Real-time Cloud Sync)
- **Authentication**: Firebase Auth + Biometric Identity Logic
- **Styling**: Tailwind CSS
- **Reporting**: PDF Invoices and Clinical Reports (jsPDF)

## Quick Start
1. **Register**: Create a patient or doctor account.
2. **Cloud Sync**: Open the same URL on your phone.
3. **Login**: Use the credentials you just created.
4. **Result**: Your data appears instantly on both screens!
